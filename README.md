# Urban Trail 

## Overview

An application where a user can see all the activities, food, shopping and weather around them with a click of a button!

## Getting Started

Install all dependencies in root

``` 
npm i 
```

Install all dependencies in client
```
cd client
npm i
```

Run app from root
```
npm run app
```
